PROJET HACKATHON GENAI - EQUIPE [VOTRE NOM]

DESCRIPTION :
Agent SRE autonome capable de prédire les pannes serveurs grâce à l'IA.

INSTALLATION :
1. Créez un environnement virtuel.
2. Installez les dépendances : pip install -r requirements.txt
3. Renommez le fichier '.env.example' en '.env' et ajoutez vos clés API.

LANCEMENT :
- Lancer l'ingestion : python ingest.py
- Lancer l'agent : python main.py
